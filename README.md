## 🛒 ClickNCart

ClickNCart is a modern**E‑Commerce Web Application** built with **React (Vite) and Redux Toolkit**. It provides a smooth shopping experience with product categories, cart management, search functionality, and a contact form powered by Formspree.



## 🛠 Tech Stack

| Layer        | Technology |
|--------------|------------|
| Frontend     | React, Vite, React Router   |
| State Management    | Redux Toolkit |
|  Form Handling |  Formspree  |
| Deployment    | Netlify  |

## ✨ Features
### 🛍️ Product Categories
- Electronics, Fashion, Beauty, Home & Kitchen
### 🛒 Cart Management 
- Add/remove items
- view totals
- dynamic price calculation
### 🔍 Search Bar 
- Search products by name with instant filtering
### 📧 Contact Form 
- Integrated with Formspree for email submissions
### 📱 Responsive UI 
- Optimized for desktop, tablet, and mobile
